JXcore samples
====================

### Run
```
  $ jx hello.js    ## start http server
  $ jx memstore.js ## test memory storage
  $ jx mt-keep: hello.js   ## keep sub threads alive
  $ jx mt-keep:4 hello.js  ## keep sub threads alive and start server four times in four threads.
  $ jx mt hello.js         ## don't keep sub threads alive
  $ jx mt:4 hello.js

  $ jx mt.js
  $ jx mt2.js
```
